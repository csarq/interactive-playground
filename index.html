Interactive Playground

Interactive Task Description

This code creates a button (btn1) that controls the background color of a div (div1) when clicked. It essentially toggles the background color between two specific shades:

  • Light blue: rgb(131, 197, 190)
  • Light pink: rgb(255, 221, 202)

Learnings from the Code

Using rgb Values Instead of Hex:

The code specifically avoids using hex code for colors (#...). This is because browsers might not always interpret and store hex colors in the same format used.

By using rgb(red, green, blue) format, the code ensures consistent color behavior across different browsers.

Toggling Background Color with Click Event:

The code utilizes an event listener (addEventListener) on the button (btn1). When the button is clicked, the script checks the current background color of div1.

Based on the current color, it sets a new background color, effectively toggling between the two defined shades.

This code snippet demonstrates a simple but effective way to interact with an element's style using JavaScript.
