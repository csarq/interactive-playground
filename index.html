Toggling an element's background colour

This code creates a button (btn1) that controls the background colour of a div (#div1) when clicked. It essentially toggles the background colour between two specific shades:

  • Light blue-green: rgb(131, 197, 190)
  • Light pink: rgb(255, 221, 202)

Learnings from this exercise

Using rgb Values Instead of Hex:

The code specifically avoids using hex code for colours (#...). This is because browsers might not always interpret and store hex colours in the same format used.

By using rgb(red, green, blue) format, the code ensures consistent colour behaviour across different browsers.

Toggling Background Colour with Click Event:

The code uses an event listener (addEventListener) on the button (btn1). When the button is clicked, the script checks the current background colour of div1.

Based on the current colour, it sets a new background colour, effectively toggling between the two defined shades.
